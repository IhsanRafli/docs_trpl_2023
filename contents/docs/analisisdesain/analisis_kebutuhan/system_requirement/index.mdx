## System Requirement
### A. User
Sistem harus memungkinkan pengguna untuk membuat akun menggunakan email atau media sosial, menyediakan fitur login/logout, pengguna juga dapat mengatur profil(nama,Alamat,kontak), system mendukung peran pengguna seperti admin,penjual, dan pembeli.

### B.Produk
Penjual dapat menambahkan,mengedit, dan menghapus produk, system bisa menampilkan daftar produk denagan informasi seperti gambar,dekripsi,harga , stok dan Alamat produk.

### C.Shopping cart
Pengguna dapat menambahkan produk ke-keranjang, system juga dapat memperbarui total harga saat item ditambahkan atau dihapus dan pengguna dapat melakukan checkout pada produk keranjang belanja.
